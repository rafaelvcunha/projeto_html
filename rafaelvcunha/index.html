Analista de Sistemas | Desenvolvedor Web | Programador Java

Sou o Rafael Vieira da Cunha, Paulistano de 25 anos, Marido dedicado, Analista de Sistemas, Desenvolvedor Java Web, Amor sobrenatural por tecnologia, Sempre em busca de conhecimento e aperfeiçoamento.

Profissional formado em Análise e Desenvolvimento de Sistemas e certificado ITIL V3 FOUNDATION, com mais de oito anos de experiência na área de Tecnologia da Informação em empresas de Serviços de EDI e Marketing.

Sólida experiência em desenvolvimento, análise e manutenção de sistemas para tradução de arquivos e troca eletrônica de dados (EDI) nos formatos XML, Edifact, Mercante, Swift, CSV e posicional. Referente aos layouts posicionais, já trabalhei com diversos padrões como Febraban, Cnab, layouts proprietário de bancos e empresas, incluindo também diversos produtos como pagamento, cobrança, tributos, extrato, débito automático, nota fiscal entre outros.

Experiência em bancos de dados MySQL, MariaDB, Microsoft SQL server e principalmente em Oracle DataBase 11g e sua linguagem PLSQL, realizando análise, tratamento e extração de dados, manutenção e desenvolvimento de Functions, Views e Procedures.

Conhecimento e experiência no desenvolvimento de aplicações com JavaSE, JavaFX, JavaEE, utilizando Hibernate JPA, JSF, CDI e EJB, Spring Framework, Spring Boot, Spring Data, Spring Batch, Maven. Desenvolvimento de webservice Rest com Json. Servidores de aplicações JBoss EAP e Wildfly, e o Container Apache Tomcat. Jenkins para integração continua.

Experiência frontend com HTML, CSS e Javascript, Frameworks Bootstrap e WordPress. Básicos Conhecimentos em desenvolvimento de aplicativos para Android com Android Studio (Java) e Xamarin (C#). Utilizei os controladores de versões CVS, SVN e GIT. Sistemas operacionais Windows e Linux, Shell Script utilizado em servidores RedHat para alterações de arquivos e análise de logs.

Rápida capacidade para aprendizagem, proativo, dinâmico e dedicado; flexível e harmonioso trabalhando em equipe; Zelo pelo prazo e qualidade na entrega de serviços.
